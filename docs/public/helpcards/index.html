siLib POP Turbulize Attribute

Lets you add noise to an attribute using one of 5 types of FBM based noise. The range of the noise is kept consistent regardless of the values of octaves and roughness. It also avoids the issues that built-in noise nodes have for input positions located around {0, 0, 0}. For details on the corresponding VEX noise functions that you can use in your own VEX code, view the siLib Turbulize VOP node. »
